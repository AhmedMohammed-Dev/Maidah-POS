# كيفية استخراج ملف EXE (البرنامج التنفيذي)

لقد تم تجهيز ملفات المشروع لتعمل كبرنامج سطح مكتب. نظراً لأننا في بيئة سحابية، لا يمكنك تحميل ملف `.exe` مباشرة من هنا. يجب عليك اتباع الخطوات التالية البسيطة على جهازك:

## الخطوة 1: تحميل المشروع
قم بتحميل ملفات هذا المشروع بالكامل على جهاز الكمبيوتر الخاص بك.

## الخطوة 2: تثبيت Node.js
تأكد من أن لديك **Node.js** مثبتاً على جهازك. إذا لم يكن لديك، حمله من: [nodejs.org](https://nodejs.org/)

## الخطوة 3: التثبيت والبناء
افتح موجه الأوامر (CMD) أو Terminal داخل مجلد المشروع، ونفذ الأوامر التالية بالترتيب:

1. **تثبيت المكتبات:**
   ```bash
   npm install
   ```

2. **بناء ملف الـ EXE:**
   ```bash
   npm run electron:build
   ```

## الخطوة 4: مبروك!
بعد انتهاء العملية، ستجد ملف التثبيت `.exe` داخل مجلد جديد اسمه **`release`** في مجلد المشروع.

يمكنك الآن نقل هذا الملف وتثبيته على أي جهاز كمبيوتر يعمل بنظام ويندوز واستخدام النظام بدون إنترنت.
