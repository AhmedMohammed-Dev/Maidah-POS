## 📝 English Description

A comprehensive, Arabic-first Restaurant & Cafe Management System designed to streamline operations from order taking to kitchen fulfillment and financial reporting. This application is built as a highly performant desktop solution using **Electron**, **React**, and **TypeScript**.

### ✨ Key Features

*   **🛒 Point of Sale (POS):** Intuitive interface for quick order processing, table management, and split-billing.
*   **🍳 Kitchen Display System (KDS):** Real-time order tracking for kitchen staff to ensure faster service and reduced errors.
*   **📊 AI-Powered Analytics:** Integrated with **Google Gemini AI** to provide smart insights into sales trends and performance.
*   **📦 Inventory Management:** Keep track of stock levels, ingredients, and product availability.
*   **👥 User & Staff Management:** Role-based access control (Admin, Cashier, Kitchen) to maintain security and accountability.
*   **📅 Daily Shifts & Accounting:** Simplified daily shift start/end procedures with automated financial reconciliations.
*   **📈 Detailed Reporting:** Visual charts and statistics using **Recharts** for sales history and revenue analysis.
*   **🌐 Arabic-First Design:** Fully supports RTL (Right-to-Left) layouts with a modern, clean UI.

### 🛠 Tech Stack

*   **Frontend:** React 18, TypeScript, Vite
*   **Styling:** Tailwind CSS (Modern & Responsive)
*   **Icons:** Lucide React
*   **Charts:** Recharts
*   **Desktop App:** Electron (Cross-platform builds)

### 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AhmedMohammed-Dev/Maidah-POS.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run in development mode:**
    ```bash
    npm run dev
    ```
4.  **Run with Electron (Desktop):**
    ```bash
    npm run electron:dev
    ```
5.  **Build for production:**
    ```bash
    npm run electron:build
    ```

## 📝 الوصف بالعربية

نظام شامل ومتكامل لإدارة المطاعم والمقاهي، مصمم باللغة العربية كأولوية لتسهيل العمليات من لحظة استقبال الطلبات وحتى تنفيذها في المطبخ وإصدار التقارير المالية. تم بناء هذا النظام ليعمل كبرنامج سطح مكتب قوي وسريع باستخدام تقنيات **Electron** و **React** و **TypeScript**.

### ✨ المميزات الرئيسية

*   **🛒 نقطة البيع (POS):** واجهة مستخدم سهلة وسريعة لمعالجة الطلبات، إدارة الطاولات، وتقسيم الفواتير.
*   **🍳 نظام عرض المطبخ (KDS):** تتبع الطلبات في الوقت الفعلي لموظفي المطبخ لضمان سرعة الخدمة وتقليل الأخطاء.
*   **📊 تحليلات مدعومة بالذكاء الاصطناعي:** مدمج مع **Google Gemini AI** لتقديم رؤى ذكية حول اتجاهات المبيعات والأداء.
*   **📦 إدارة المخزون:** تتبع مستويات المخزون، المكونات، وتوافر المنتجات بسهولة.
*   **👥 إدارة المستخدمين والموظفين:** نظام صلاحيات متطور (مدير، كاشير، مطبخ) لضمان الأمان والمحاسبة.
*   **📅 الورديات اليومية والمحاسبة:** إجراءات مبسطة لبدء وإقفال الوردية اليومية مع تسويات مالية تلقائية.
*   **📈 تقارير تفصيلية:** رسومات بيانية وإحصائيات بصرية باستخدام **Recharts** لتاريخ المبيعات وتحليل الإيرادات.
*   **🌐 تصميم عربي أصيل:** دعم كامل لواجهات اليمين إلى اليسار (RTL) مع تصميم عصري وجذاب.

### 🛠 التقنيات المستخدمة

*   **الواجهة الأمامية:** React 18, TypeScript, Vite
*   **التنسيق والتصميم:** Tailwind CSS (تصميم عصري ومتجاوب)
*   **الأيقونات:** Lucide React
*   **الرسوم البيانية:** Recharts
*   **تطبيق سطح المكتب:** Electron
